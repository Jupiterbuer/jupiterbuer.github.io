JavaScript的数据类型

JavaScript数据类型

JavaScript中什么是基本数据类型？

基本数据类型：

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Symbol
  • biglnt

JavaScript中什么是引用数据类型？

引用数据类型统称为Object
细分：

  • Object
  • Array
  • Date
  • Function
  • RegExp

JavaScript中各个类型的数据如何储存？

1、基本数据类型直接存储在栈中；
2、引用数据类型存储在堆中，在栈中保存数据的引用地址，此引用地址指向对应的数据，以便快速查找到堆内存中的对象。
注：栈内存自动分配内存，而堆内存是动态分布内存，不会自动释放。故每次使用完对象的时候都要将它设置为nul，从而减少无用内存的损耗。

HTML5新特性

HTML5新特性

HTML5 是最新的 HTML 标准。

HTML5 是专门为承载丰富的 web 内容而设计的，并且无需额外插件。

HTML5 拥有新的语义、图形以及多媒体元素。

HTML5 提供的新元素和新的 API 简化了 web 应用程序的搭建。

HTML5 是跨平台的，被设计为在不同类型的硬件（PC、平板、手机、电视机等等）之上运行。

弥补以前的不足，增加新的标签、新的表单和新的表单属性等。

优势：不考虑兼容性的问题可以尽量使用新特性，优化代码。
