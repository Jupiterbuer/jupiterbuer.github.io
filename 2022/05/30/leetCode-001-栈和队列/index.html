栈和队列

在 JavaScript 数组中，我们可以通过索引随机访问元素，但是在某些情况下，我们可能要限制数据的访问顺序，于是有了两种限制访问顺序的数据结构：栈（stack）（先进后出）、队列（queue）（先进先出）。

包含 min 函数的栈

题目要求

定义栈的数据结构，请在该类型中实现一个能够得到栈中所含最小元素的min函数（时间复杂度应为O（1））。

示例

算法思路

利用辅助栈来实现，辅助栈中存放最小值。

  1. 定义两个栈，一个用于存储数据，另一个栈（辅助栈）用于存储每次进栈的最小值。
  2. 数据每次进栈时，和辅助栈栈顶元素进行比较，讲两者比较的最小值再次压入栈。
  3. 数据出栈，辅助栈也出栈。

滑动窗口最大值

题目描述

给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

示例

算法思路

找最大值和最小值都可以通过辅助队列实现。

此题寻找滑动窗口最大值：

  • 模拟一个窗口
  • 在寻找当前窗口中的最大值
  1. 模拟窗口，窗口右边为right左边为left
  2. 右窗口依次递增
  3. 左窗口为right - k + 1
  4. 当队列不为空且当前元素大于等于队列尾部元素，则将队列尾部元素删除，再将当前元素放入队列。
  5. 判断窗口开始的条件right+1 >= k时，将队列首部元素放入结果数组中。

用两个栈实现队列

题目描述

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

算法思路

  1. 将一个栈当作输入栈，用于压入 appendTail 传入的数据；另一个栈当作输出栈，用于deleteHead操作。
  2. 每次deleteHead 时，若输出栈为空则将输入栈的全部数据依次弹出并压入输出栈，这样输出栈从栈顶往栈底的顺序就是队列从队首往队尾的顺序。

栈的压入和弹出序列

题目描述

输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如，序列 {1,2,3,4,5} 是某栈的压栈序列，序列 {4,5,3,2,1} 是该压栈序列对应的一个弹出序列，但 {4,3,5,1,2} 就不可能是该压栈序列的弹出序列。

示例：

算法思路

  1. 借助辅助栈来模拟压入和弹出的过程
  2. 设置一个索引index记录poped(出栈序列)栈顶的位置
  3. 将pushed中的数据依次压入辅助栈
  4. 当辅助栈栈顶元素和popped栈顶元素相同时，则辅助栈出栈，index++
  5. 出栈有可能在任意一次入栈后进行，当辅助栈栈顶元素和压popped栈顶元素相同时，继续让pushed入辅助栈
  6. 所有操作完成，若辅助栈为空，则出栈顺序正确。
