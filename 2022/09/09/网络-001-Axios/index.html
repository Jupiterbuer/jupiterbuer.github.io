Axios

Axios 基于promise 的 HTTP 库。

特点

  • 浏览器发起 XMLHttpRequest 请求
  • 支持 Promise API
  • 拦截请求
  • 对请求和返回进行转化
  • 取消请求
  • 自动转换为 json 数据格式
  • 支持抵御 XSRF 攻击

拦截器

axios 提出响应拦截器和请求拦截器来处理相关的响应和请求。

  • 请求拦截器：在请求发送之前统一执行某些操作，如给在请求头中田间 token 字段。
  • 响应拦截器：在收到服务器的响应时执行某些操作，如发现响应状态码为 401 （请求需要通过HTTP认证）时自动跳转到登录页面。

Axios 作用在于发送 HTTP 请求，对于请求和响应拦截器而言，实际上便是一种具有特定功能的函数。

拦截器的实现思路

  • 任务注册：注册任务，注册的任务会把方法存储某个内部数据结构之中。
  • 任务编排：注册的发送请求前的配置 => 发送HTTP请求 => 处理响应请求

​ 使用 unshift 和 push 方法分别插入到执行队列中

  • 任务调度：对 chain 中的任务取出，利用 Promise 实现

CSRF

CSRF：跨站请求伪造

防治措施：

  • 检查 Referer 字段
  • 同步表单校验
  • 双重 Cookie 防御（axios基于此实现）
