HTML5新特性

HTML5 是最新的 HTML 标准。

HTML5 是专门为承载丰富的 web 内容而设计的，并且无需额外插件。

HTML5 拥有新的语义、图形以及多媒体元素。

HTML5 提供的新元素和新的 API 简化了 web 应用程序的搭建。

HTML5 是跨平台的，被设计为在不同类型的硬件（PC、平板、手机、电视机等等）之上运行。

弥补以前的不足，增加新的标签、新的表单和新的表单属性等。

兼容性问题：基本需要**IE9+**以上的浏览器才支持。

优势：不考虑兼容性的问题可以尽量使用新特性，优化代码。

1. 新增语义化标签

2. 新增多媒体标签

  • 音频标签
    支持三种格式：MP3、Wav、Ogg
  • 视频标签
    支持三种格式：MP4、WebM、Ogg
