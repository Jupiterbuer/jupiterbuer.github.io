CSS盒子模型

一、盒子模型

对一个文档进行布局时，浏览器渲染引擎会根据 CSS 基础盒模型，将所有元素表示为一个个矩形盒子。

盒子由四部分组成：

  • content：实际内容，显示文本图像
  • padding：内边距，内边距是透明的，受盒子的background影响
  • border：围绕内边距的一条绘者多条线段，由粗细、样式、颜色三部分组成
  • margin：外边距，元素周围创建额外的空白

二、标准盒模型

CSS中盒子模型可分为：

  • W3C标准盒模型
    • 盒子总宽度 = width + padding + border + margin
    • 盒子总高度 = height + padding + border + margin
  • IE 盒模型
    • 盒子总宽度 = width + margin
    • 盒子总高度 = height + margin

三、box-sizing属性

box-sizing属性：定义引擎如何计算一个元素的gaodu

box-sizing:content-box:元素的宽和高不包含padding和border，标准盒子模型

box-sizing:border-box:元素的宽和高包含padding和border，IE盒子模型

box-sizing:inherit:继承父元素的属性
