JavaScript数据类型

JavaScript中什么是基本数据类型？

基本数据类型：

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Symbol
  • biglnt

JavaScript中什么是引用数据类型？

引用数据类型统称为Object
细分：

  • Object
  • Array
  • Date
  • Function
  • RegExp

JavaScript中各个类型的数据如何储存？

1、基本数据类型直接存储在栈中；
2、引用数据类型存储在堆中，在栈中保存数据的引用地址，此引用地址指向对应的数据，以便快速查找到堆内存中的对象。
注：栈内存自动分配内存，而堆内存是动态分布内存，不会自动释放。故每次使用完对象的时候都要将它设置为nul，从而减少无用内存的损耗。
